# Sales & Revenue Analytics Dashboard | Power BI

![Dashboard Preview](dashboard_overview.png)

An end-to-end business analytics solution designed to analyze retail sales trends, category and regional performance, and customer profitability. Includes forecasting to enable strategic decision-making.

---

## Tech Stack
- Power BI
- Power Query
- DAX
- Excel
- ETL
- Forecasting
- Data Modeling

---

## Features
- KPI dashboard with Total Sales, Profit, Profit Margin %, YOY Growth
- ETL pipeline built using Power Query
- Star Schema with DimDate & FactSales
- Trend analysis & forecasting
- Insights & recommendations for business planning

---

## Key Insights
- New York contributes the highest sales across all cities.
- Technology category drives majority of revenue while Copiers produce highest profit.
- Sales increased consistently from 2014 to 2018.
